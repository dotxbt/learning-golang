# Learning Golang
Go Programming Language

#### References : 
- [Golang Documentation](https://golang.org/doc/)

#### Install in Ubuntu 
```
sudo apt install golang
```

#### init Project Folder
```
go mod init learning-golang
```

#### Run Main Application
```
go run Main.go
```

#### Build Executable
```
go build Main.go
```

#### Run Executable
```
./Main
```

#### Clean Project
```
go clean
```

#### Install to local go bin
```
go install Main.go
```
